# Fiscal-Flow-Application

### Objective

To develop a client-facing personal finance web application that helps users track income and expenses efficiently, while applying full-stack development skills with Python, Flask, HTML/CSS, and JSON-based data management.

### Skills Learned

- Full-stack web development with Python and Flask
- Frontend design using HTML and CSS for intuitive user interfaces
- Data management with JSON for persistent storage
- Building client-facing applications to meet real user needs
- Collaborative development and teamwork during project planning and implementation
- Problem-solving and debugging in a live project environment
  
### Tools Used

- Python & Flask
- HTML & CSS
- JSON data storage
- VSCode
- Client Feedback
